# 參團須知

各位如果要參加交期比較長的團，參加前請自行評估可能的風險。

常見的風險包括但不限於以下

- 截止後，又看到更便宜的貨源
- 截止後，過了一段時間，因任何原因而流團
- 等了很久，因到貨數量不足而被砍單
- 到貨的時候市場價格跳水了
- 到貨的時候已經不喜歡了
- 貨幣匯率變動（通常不會太大，今年比較特別）

無論是什麼原因，從我的角度而言，只要未依當初的約定價買回，那麼就是跑單。各位要怎麼處理到手的物品我無權干涉，甚至要轉賣賣高我也不介意（我會認為那是你的本事），或是找得到人轉單我直接出貨給對方也可以。

希望大家參團前能知曉以上常見的風險以及評估個人的風險承受能力，感謝 🙏
